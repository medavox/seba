Space Engineers Blueprint Analyser

This simple web app creates a breakdown of various important stats in the supplied blueprint.

It needs a copy of the game's data files, namely sbc recipes.xml file (which contains all the crafting recipes in the game), as the game license prohibits including it with this app.

A benefit of you supplying your own copy is that the crafting recipes will always be for your version.

Your recipes.xml file can usually be found in: 

C:\Program Files (86)\Steam\steamapps\common\7 Days To Die\Data\Config\
(You can copy and paste this into the file picker)
